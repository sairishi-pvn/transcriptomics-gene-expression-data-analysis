# Transcriptomics Gene Expression Data Analysis

This repository contains workflows for gene expression analysis using
RNA-seq and microarray data.

## Scope
- RNA-seq differential expression analysis
- Microarray (Affymetrix) gene expression analysis
- Data normalization, filtering, and visualization
- Statistical testing using Bioconductor tools

## Tools & Technologies
- R
- Bioconductor
- edgeR
- limma
- affy

## Purpose
This repository serves as a learning and portfolio project for
transcriptomics and bioinformatics data analysis.
